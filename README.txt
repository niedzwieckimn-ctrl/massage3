
Massage & SPA — Static v5 (LocalStorage)
----------------------------------------

To jest lekka, statyczna wersja aplikacji rezerwacyjnej z panelem admina.
Dane są przechowywane w localStorage przeglądarki (na czas testów / prototypu).

Pliki:
- index.html      — panel klienta (rezerwacje)
- admin.html      — panel admina (PIN domyślny: 2505)
- assets/style.css, store.js, app.js, admin.js, logo.svg

Funkcje:
- Edycja usług/cennika
- Dodawanie/usuwanie wolnych terminów
- Brak overbookingu (zajęty slot znika z listy)
- Lista rezerwacji z potwierdzaniem
- Lista klientów, profil, notatki, preferencje, sugestie terapeutyczne
- Ustawienia (kontakt, RODO, PIN)
- Potwierdzenie mailem (mailto:)
